## Fgt X-Ray
Minecraft Forge XRay mod

## Feature:
    - Awesome GUI
    - Customizable search radius
    - Fancy detection render
    - Able to add/remove/edit ores/blocks
    - Color selector (RGB)
    - Hotkeys to enable/disable options
